# Assesment 2 – Struktur Data 2025/2026

Repository ini berisi implementasi berbagai struktur data dasar menggunakan C++, seperti **Stack**, **Queue**, sehingga mudah dipahami oleh mahasiswa praktikum.

---

## Cara Download (Git Clone)

Pastikan Git sudah terpasang di komputer Anda.  
Untuk mendownload seluruh repository, jalankan perintah berikut:

```sh
git clone https://github.com/Codex2142/Assesment-2-Struktur-Data-25-26.git
```

## Cara Compile

masuk ke direktori `Stack` atau `queue` dengan cara dibawah ini:
```sh
cd queue
```
kemudian compile kode program dengan cara dibawah ini:
```sh
g++ main.cpp queue.cpp -o main
```

## Cara Run
jalankan file `main` dengan cara dibawah ini:
```sh
./main
```
